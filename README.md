Lagoa Parque Hotel - Cardápio Digital

Projeto desenvolvido para a disciplina de Extensão Universitária.

Objetivo

Desenvolver uma solução digital para disponibilização do cardápio do restaurante do Lagoa Parque Hotel através de QR Code, proporcionando maior praticidade aos hóspedes e facilitando a atualização das informações.

Tecnologias utilizadas
HTML5
CSS3
JavaScript
GitHub Pages
Funcionalidades
Cardápio digital responsivo
Filtro por categorias
QR Code para acesso rápido
Compatível com dispositivos móveis
